# Starbucks Project ☕

Um projeto de landing page responsivo do Starbucks, destacando promoções e ofertas especiais de bebidas.

## 📋 Descrição

Este é um projeto frontend que recria a interface de uma landing page do Starbucks, com foco em apresentar as ofertas de Happy Hour e produtos exclusivos. O projeto utiliza HTML5, CSS3 e é totalmente responsivo.

## 🎯 Funcionalidades

- ✨ Design moderno e responsivo
- 🔍 Barra de pesquisa funcional
- 📱 Menu responsivo
- 🎨 Galeria de imagens dos produtos
- 💬 Seções com citações inspiradoras
- 🏪 Cards com promoções e ofertas

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização e layouts responsivos
- **Responsive Design** - Adaptável a diferentes dispositivos

## 🚀 Como Usar

1. Clone o repositório:
```bash
git clone https://github.com/LucasFerreiraDaSilva/Starbucks-Project.git
```

2. Navegue até o diretório do projeto:
```bash
cd Starbucks-Project
```

3. Abra o arquivo `index.html` no seu navegador favorito

## 📱 Responsividade

O projeto foi desenvolvido com abordagem mobile-first e é totalmente responsivo para:
- Dispositivos móveis (smartphones)
- Tablets
